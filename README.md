AI College Football Playoff Predictor and Simulator

Overview
AI driven project to predict college football playoff outcomes using historical data and machine learning models.

How To Run:
1. pip install -r requirements.txt
2. streamlit run app.py

Features
1. Multiple ML models (Logistic Regression, Random Forest, KNN)
2. Monte Carlo playoff simulation
3. Adjustable chaos (randomness)
4. Visualizations and bracket

Dataset sourced from Kaggle.
Models predict team strength and use that to drive simulation.
